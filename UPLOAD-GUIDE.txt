METHOD 1: FTP/SFTP CLIENT (Easiest)
====================================
1. Open FileZilla, Cyberduck, or your FTP client
2. Connect to: www.simongrant.org
3. Navigate to your web directory
4. Drag and drop the 4-6 files listed above
5. If using server storage, create a "data" folder and set permissions to 755


METHOD 2: COMMAND LINE (if you have SSH)
=========================================
From your terminal, run:

cd /home/user/2026-Claude-trial

scp index.html app.js question-data.js styles.css api.php \
  your-username@www.simongrant.org:/path/to/web/directory/

Then SSH to your server and create the data directory:
ssh your-username@www.simongrant.org
mkdir -p /path/to/web/directory/data
chmod 755 /path/to/web/directory/data
